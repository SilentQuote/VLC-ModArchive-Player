# VLC-ModArchive-Player
An extension for VLC media player that automatically streams a random module from the ModArchive.

## Installation
- Download from the Releases tab or from [VLC's official extension website](https://www.pling.com/p/2349912/)
- [Follow the official guide for installing LUA extensions](https://vlc-user-documentation.readthedocs.io/en/latest/addons/extensions.html#installing-extensions-in-lua) (at least until it starts showing up in the built-in extension menu...)
